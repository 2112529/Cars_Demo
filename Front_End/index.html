About this project

项目介绍

本案例旨在利用纽约汽车数据进行深入数据分析和可视化展示，以揭示与纽约市汽车相关的有价值的信息。数据集包括多个字段，涵盖了汽车的状态、名称、价格、外观颜色、内饰颜色、驱动方式、燃油效率、燃料类型、传动方式、发动机、里程数、便利性、娱乐性、外观、安全性、座椅、清洁标题、品牌、年份、型号、总体评分、舒适度评分、设计评分、性价比评分、风格评分和可靠性评分。

通过对这些数据的分析和可视化展示，分析得到：汽车市场趋势、汽车属性关联性、燃料效率和驱动方式、汽车价值和风格等。
